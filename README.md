
Each assignment folder contains individual C programs named as `Q01.c`, `Q02.c`, etc., arranged in proper order.

---

##  Assignment 1: Basics of C Programming

1. Write a C program to declare variables of type `int`, `float`, `char`, and `double`.  
   Initialize them with values and print their values and sizes using the `sizeof` operator.

2. Write a C program to input two integers and perform:
   - Addition  
   - Subtraction  
   - Multiplication  
   - Division  
   - Modulus  

3. Write a C program to demonstrate the difference between **pre-increment (++x)** and  
   **post-increment (x++)** in an arithmetic expression.

---

##  Assignment 2: Operators and Decision Making

1. **Salary Calculation System**  
   Write a C program that:
   - Accepts employee name and basic salary  
   - Calculates:
     - HRA = 20% of basic salary  
     - DA = 10% of basic salary  
     - Gross Salary  
     - Income Tax = 5% of gross salary  
     - Net Salary  
   - Displays all salary details

2. **Comparing Two Numbers Without Conditional Statements**  
   Write a C program that:
   - Takes two account balances as input  
   - Uses relational and logical operators only  
   - Displays results as `1 (True)` or `0 (False)`

3. **Login Authentication System**  
   Write a C program that:
   - Stores predefined username and password  
   - Validates user input  
   - Displays “Login Successful” or “Invalid Login”

4. **Bank Interest Update System**  
   Write a C program using assignment operators:
   - `+=`, `-=`, `*=`, `/=`  
   - Displays updated balance after each operation

##  Assignment 3: Looping and Pattern Programs

1. Write a C program to print **A to Z alphabets** using a **star (*) pattern**.

2. Write a C program to print **your name** using a **star (*) pattern**.

##  Assignment 4: Functions and Pointers

1. Write a C program that uses a **user-defined function** to swap two numbers using  
   **call by reference**.

2. Write a C program to find the **maximum of two numbers** using a function that accepts  
   **pointer arguments**.

---

##  Assignment 5: Arrays

1. Write a C program to read **5 integers** into an array and print them.

2. Write a C program to store **n numbers** in an array and find their **sum**.

3. Write a C program to **delete an element** from a specified position in an array.

4. Write a C program to **search an element** in an array using **linear search**.

5. Write a C program to **sort array elements** in **ascending order**.
